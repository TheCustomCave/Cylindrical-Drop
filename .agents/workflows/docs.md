# Documentation Workflow

When triggered, perform the following steps:

1. **Scan APIs:** Check `gameStore.ts` and utility files for undocumented public functions.
2. **Generate JSDoc:** Add or update JSDoc comments for complex logic.
3. **Update README:**
   - If new game settings or controls are added, update the "Controls" or "Developer Guide" sections.
   - Ensure the "Tech Stack" section reflects any new dependencies.
4. **Consistency Check:** Ensure `GEMINI.md` reflects the current project structure.
5. **Output:** List all documentation changes made.
